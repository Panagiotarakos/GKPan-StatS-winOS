## Features
- CPU usage monitoring
- RAM usage monitoring
- Disk usage monitoring
- Network speed (download/upload)
- Battery status

## Install
1. Download ZIP from GitHub
2. Open `GKPanStats.sln` in Visual Studio
3. Build & Run (F5)
